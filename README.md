# Rubber Duck
###The Rubber Duck Debugging App!

What is rubber duck debugging? Read [here](http://www.rubberduckdebugging.com/)!

What this rubber duck can do:
  1. It can blink
  2. It can follow your mouse (simple)
  3. It can help you with your bugs
